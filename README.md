### `Run Locally`

+ Clone the project:

      git clone https://github.com/satishshinde2048/TodoApp-Functionbased.git


+ Navigate to the project directory:

      cd TodoApp-Functionbased
                                                                                                       
+ Install dependencies:                                                                                                    

       npm install 
                                                                                                                                                                                                                
+ Start the server:                                                                                                        

       npm start
